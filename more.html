    This Website

    After learning HTML and CSS on codecademy.com, I wanted to put theory into practice. Since this was just a small side project of mine, I opted to use github as a host for the site instead of buying my own domain. It was a little tough at first, but I feel more comfortable with HTML and CSS as a result of this project. As a novice CSS programmer and as someone who enjoys designing things, I am proud of how it turned out.
